# Source directory – agents

- **Blog posts**: Add markdown files with YAML front matter in `source/blog/` using the filename pattern `YYYY-MM-DD-slug.html.markdown`. Required front matter: `title`, optional: `date`, `description`, `social_image`, `social_image_alt`. Posts are rendered by middleman-blog and appear automatically in the Writing section on the home page.
- **Writing section & blog index**: Use the `merged_blog_posts` helper (defined in config.rb `helpers do`) for a single list of local articles + `data.blog` entries, merged and sorted by date (newest first). Render with `partial "partials/blog_post_card"` and `locals: { entry: entry }`; each entry is `{ date:, local: true/false, article: or post: }`. External posts link out (target="_blank"); local posts use relative URLs.
- **Blog index**: With `blog.prefix = "blog"` and directory_indexes, the blog index is `source/blog/index.html.erb`; it is served at `/blog/`. Use the same section/card layout as the home Writing section; both use `merged_blog_posts` and `_blog_post_card` for consistent, date-sorted display.
- **Blog layout**: Articles use `blog_layout` (set in config.rb). The layout uses `current_article` to show article title and date in the page header; inner pages use `partial "partials/nav"` for Home and Blog links.
- **Sticky footer**: Body uses `min-height: 100vh; display: flex; flex-direction: column`. Main content (e.g. `.page-content`, `.blog.page-content`) has `flex: 1 1 auto` so it fills space; footer has `flex-shrink: 0` and minimal padding so it stays at the bottom and takes minimal height.
